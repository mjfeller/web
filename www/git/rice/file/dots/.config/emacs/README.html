      1 Installation
      2 
      3 To install, clone this repo to ~/.emacs.d.
      4 
      5         git clone https://gitlab.com/mark.feller/emacs.d ~/.emacs.d
      6 
      7 Upon starting up Emacs for the first time, packages will be downloaded and
      8 installed via the public repos. If you encounter any errors at that stage, try
      9 restarting Emacs, and possibly running 'M-x package-refresh-contents' before
     10 doing so. After successfully installing packages future startups will be much
     11 faster.
     12 
     13 There are several subdirectories:
     14 
     15 'core'         holds primary configuration that I feel lost without.
     16 'elpa'         holds vendored packages from package.el.
     17 'etc'          holds configuration files for packages, files often written to
     18                like bookmarks.el, and the custom.el file used when configuring
     19                emacs through the customization interface.
     20 'lisp'         holds custom user lisp code. Useful for personal packages,
     21                personal information, etc.
     22 'modules'      holds package/language specific configuration that is useful when
     23                working with specific tasks or within specific languages, but are
     24                not needed generally.
     25 'src'          holds the Emacs source directory for jumping to definition and
     26                viewing docs within the C code.
     27 'snippets'     holds Yasnippet snippets.
     28 'theme'        holds theme packages. Poet is my current goto.
     29 
     30 Disclaimer
     31 
     32 This is an everchanging and evolving Emacs configuration for my personal use. I
     33 see this repo more as a collection of ideas to be incorporated into your own
     34 configurations.
