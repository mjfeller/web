      3 # kubel
      4 
      5 Emacs extension for controlling Kubernetes with limited permissions.
      6 
      7 ## Features
      8 This is mostly pod management for now. More may come.
      9 
     10 - switch context and namespace
     11 - show pods
     12 - get pod details
     13 - tail container logs (possibly with `-f` follow flag)
     14 - copy pod name to clipboard
     15 - copy container log command to clipboard
     16 - port forward a pod to your localhost
     17 
     18 ## Installation
     19 
     20 Get it from Melpa, or copy and load the `kubel.el` file.
     21 
     22 ## Usage
     23 
     24 To list the pods in your current context and namespace, call
     25 ```
     26 M-x kubel
     27 ```
     28 To set said namespace and context, respectively call
     29 ```
     30 M-x kubel-set-namespace
     31 M-x kubel-set-context
     32 ```
     33 Note that namespace will autocomplete but not context,
     34 this is because I interact with kubernetes through a user who
     35 does not have permissions to list namespaces.
     36 
     37 ## Shortcuts
     38 
     39 On the kubel screen, place your cursor on the pod
     40 ```
     41 enter => get pod details
     42 h => help popup
     43 C => set context
     44 n => set namespace
     45 g => refresh pods
     46 p => port forward pod
     47 e => exec into pod
     48 d => describe popup
     49 l => log popup
     50 c => copy popup
     51 k => delete pod
     52 j => jab deployment to force rolling update
     53 ```
     54 
     55 ## Customize
     56 
     57 By default, kubel log tails from the last 100 lintes, you can change the `kubel-log-tail-n` variable to set another line number.
